Place rain.wav and thunder.wav here.

Free sources:
  https://freesound.org/search/?q=rain+loop     (use a 30+ second loop for rain.wav)
  https://freesound.org/search/?q=thunder       (a single crack for thunder.wav)

Both files must be WAV format.
RainDelay will work without them — sound is simply skipped if files are absent.
